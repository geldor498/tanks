Папку libs нужно прописать в путях или проекта (если это MSVC 2010 и более) или в путях студии. 
Нужно лишь прописать данную папку в списке каталогов include.
Папка проектов tanks. 
.\tanks\clientc\Release\ содержит исполняемый файл простейшего клиента и батник для демонстрации того, какие параметры нужно задавать для запуска.
.\tanks\server\Release\ сожержит исполняемый файл сервера.

в начале запустите сервер (.\tanks\server\Release\servapp.exe) 
затем запустите клиета (.\tanks\clientc\Release\clientc.bat)
для начала работы сервера необходимо в меню выбрать Game\Start
после этого физический движок начнёт обрабатывать команды поступающие с клиента.
для просмотра информации о текущей ситуации можно выбрать View\Information 
отобразиться окно с суммарным количеством жизней у каждой команды и с информацией о танках.
